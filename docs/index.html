# Iniciando el desarrollo

# Requerimientos de la web

Se solicita la creación de un sitio web sobre publicaciones de posts o artículos de diferentes tópicos que cuente con un registro y login de usuarios. Los usuarios no registrados podrán ver los posts publicados agrupados por categorias. Solo usuarios que esten loggeados tendrán la atribución de crear posts y crear categorías nuevas. Un post contiene un titulo, una descripción, una categoría, una imagen, el usuario que lo publicó y la fecha en la que fue creado. Los usuarios cuentan con un perfil en el que pueden editar sus datos personales y ver los posts que han realizado, junto con la posibilidad de editarlos y eliminarlos, el perfil también cuenta con dos formularios, uno para crear posts nuevos y el otro para crear categorías nuevas.

Información

Este proyecto está basado en el curso Master en PHP, sin embargo, se agregaron funcionalidades extra al proyecto propuesto originalmente por el instructor.

# Conexión a base de datos

Se crea archivo con nombre connection.php en el cual se instancia un objeto mysqli para su posterior importación e utilización en los archivos que requieran una conexión a la base de datos.

<?php
    $servername = 'localhost';
    $username = 'fromiti';
    $password = 'admin';
    $database = 'posts';
    $conn = new mysqli($servername, $username, $password, $database);
    // Check connection
    if ($conn->connect_error) {
      die("Connection failed: " . $conn->connect_error);
    }else{
        session_start();
    }
